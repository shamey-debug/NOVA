# KoolCrypto Starter

A simple Next.js starter for a crypto learning platform.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## What's included

- Homepage
- Signup page
- Dashboard page
- Lesson page
- Simulator page
- Pricing page
- Plain CSS styling
- No shadcn dependency
- Easy to connect to Supabase later
